Javascript es un lenguaje de programación de tipado dinámico que tiene muchas referencias de otros lenguajes dentro de él, como por ejemplo en la forma de escritura, su sintaxis y la forma de operación. Se ejecuta sobre los navegadores, pero también en una gran variedad de entornos y dependiendo del entorno la forma en la que se programa Javascript es distinta. Se utiliza en frontend, backend, aplicaciones móviles, entre otras opciones. JavaScript fue diseñado con una sintaxis similar a Java para que los programadores que estuvieran familiarizados con Java pudieran aprenderlo más fácilmente. Ambos lenguajes son orientados a objetos. Aún así, JavaScript y Java son lenguajes distintos con diferentes características, propósitos y entornos de ejecución.
